# 🎬 Movie Recommender System

A simple machine learning project that recommends movies based on similarity using TF-IDF and cosine similarity.

---

## 🚀 Features
- Enter a movie name
- Get top 5 similar movies
- Web interface using Flask

---

## 🛠 Technologies Used
- Python
- Flask
- Pandas
- Scikit-learn

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
python app.py
